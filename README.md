## eNotes ≽(•⩊ •マ≼
Encrypted notes for every purpose

Have you ever wanted to save notes but every single app never actually protected the contents? eNotes is the no-bullshit alternative to all of them.

Written in C++, eNotes protects the contents of your files using AES256 encryption (Which is used by the US government). You can use whatever key you want: text, images, audio, you name it.

## Usage ˖˚⊹ ꣑ৎ‎

eNotes at its core is a command line tool. We hope to expand it into a full TUI interface in the future. 

To encrypt a file (which currently only accepts .md and .txt), run the command +c <file> <string/file>. This creates a .enote file in the old files place.

To decrypt a file, run the command -c <file> <string/file>

For a more in-depth manual on each command, refer to the wiki.

## What's Different ≽(•⩊ •マ≼

eNotes is different from other "privacy focussed" note apps because **nothing** is stored online. 

Not even the app itself *knows* that it just encrypted that file. To it, the file could've been from your friend, the internet, or a USB stick in a random wall...
